
# Shop Bridge

A web application to keep record of products in your inventory.




## Run Locally



Clone the project

```bash
  git clone https://github.com/abhaysinghs772/shop_bridge.git
```

Go to the project directory

```bash
  cd shop_bridge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node src/app.js
```

  
## Tech Stack

**Client:** Ejs, css

**Server:** Node, Express, mongoDB
